## Description

Unidux を利用した状態制御・画面遷移機構のサンプルプロジェクトです．</br>
UniDi は Zenject をリファクタリングしたリポジトリで，実質 Zenject と同じです．

## Dependencies

* Unity 2021.2.5f1
